# Bootstrap Plan

## Chosen repo name

`x-bookmark-knowledge-pack`

This name emphasizes:

- X bookmarks
- portable packaging
- knowledge-pack framing

instead of a generic “viewer only” positioning.

---

## Immediate next steps

1. bring over only public-safe viewer code from the private repo
2. define sample data format
3. define packaging command/output shape
4. add a public-friendly setup flow
5. make the first release around static artifacts, not cloud infra

---

## First public milestone

The first milestone should be:

**Generate a portable local bundle from sample bookmark data**

including:

- gallery HTML
- tags JSON
- translations JSON
- README

---

## Non-goals for first milestone

- cloud sync
- hosted multi-user product
- full search competition with bookmark SaaS tools
- over-engineered frontend rewrite
